 exercício feito durante algumas aulas de html, css e javascript do curso de introdução a desenvolvimento web do instituto nu by descomplica.

 fiz uma alteração que não está no site Replit que foi uma condição. caso não fosse colocado nenhum nome, o site daria um alerta para que fosse colocado.
 antes disso o site aceitava a possibilidade de não terem escrito nada e apresentar uma mensagem de boas-vindas.
 exemplo: "Bem vindo, !"
 agora com essa condição, o site só exibirá uma mensagem de boas-vindas se tiverem colocado algum nome.
